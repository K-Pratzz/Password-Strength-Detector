PASSWORD STRENGTH  DETECTOR

RECKLESS PASSWORD HABITS :

    
    
        
  1. Using personal information : 
        Passwords based on your name, birthday, 
        pet's name, or other public details are easy for hackers to guess.
    2. 
        
  2. Reusing passwords : 
        Using the same password for multiple accounts is a major risk. 
        If one service is compromised, all of your accounts are at risk.
    3. 
        
  3. Simple, common words : 
        Passwords like "password," "123456," or "qwerty" are the first ones 
        hackers and their tools will try.
    4. 
        
  4. Short passwords : 
        Passwords that are too short are easy to crack.
        The longer a password is, the more difficult it is for a computer to guess.

SUGGESTIONS FOR A STRONG PASSWORD :

    
    
    
  1.  Use a password manager : 
    This is the easiest and most secure way to manage your passwords.
     A password manager can create and store long, complex, 
     and unique passwords for all your accounts.
    2. 
    
  2.  Aim for length :
    Greater the number of suitable characters, higher the reliability.
    A good password should be at least 12 characters long.
    3. 
    
  3.  Combine different types of characters : 
    Use a mix of uppercase letters, lowercase letters, numbers, and 
    special characters (like !, @, #, etc.) to make your password harder to guess.
    4. 
    
  4. Create a passphrase :
     Instead of a single word, think of a memorable sentence or phrase and
     use the first letter of each word to create a unique password. 
     For example, "My dog's name is Rex" could become "Mdnir@1!"
